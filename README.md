#  YouTube Mashup Generator

A full-stack Python application that generates a mashup by:

- Downloading YouTube videos of a singer
- Extracting audio from each video
- Trimming the first N seconds
- Merging audio clips
- Sending the final ZIP file via Email

Deployed using Flask + Gunicorn on Render.
